# Proyecto de Visión por Computadora 
- Se reentrenó el modelo YOLO Nano 2026 para detectar frutas en tres estados de maduración: sin madurar, maduro y pasado. 
- Se realizó un Benchmark para comparar los tiempos de entrenamiento en CPU vs. GPU.
- Entorno utilizado: Google Collab - GPU T4.
